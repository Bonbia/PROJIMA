Ce Git resume l'entiereté du projet Comparaison de deux méthodes non locales de débruitage d’images réalisé par Adam Fayret--Hochart et encadré par Elsa Angelini.

Les deux fichiers finaux sont BM3D_clean.ipynb et NLM_clean.ipynb tout les resultats finaux proviennent de ces deux fichiers qui peuvent donc être fait tourner sans problèmes.
Les resultats sont dans le fichier resultat pris d'image source du dossier img.
